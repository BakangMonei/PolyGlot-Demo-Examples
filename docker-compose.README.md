# docker-compose.yml (optional local lab)

The root `docker-compose.yml` file is an **optional lab sketch** for readers who want to stand up MySQL, Flyway migrations, Kafka, Jaeger, and a sample gateway + service chain. It is **not** a production topology and does not replace your own environment definitions.

For a documentation-only reading path, you can ignore this file entirely and follow [BUILDERS_GUIDE.md](./BUILDERS_GUIDE.md).
